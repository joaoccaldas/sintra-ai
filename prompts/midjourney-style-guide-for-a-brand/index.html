Midjourney Style Guide for a Brand

A reusable Midjourney style guide with core suffix, 5 style references, and brand-safe guardrails.

⬡
Recommended modelGPT-5.5 Instant

Best visual reasoning for creative direction

What you need to fill in

[brand name][list 3 values][describe audience]

Tools used

MidjourneyNotion

The prompt

Create a Midjourney style guide for [brand name] with the following brand values: [list 3 values] and target audience: [describe audience]. Include: a core prompt suffix all images should use, 5 approved style references with example prompts, 3 things to always avoid, and a color palette description to include in prompts.
Open in ChatGPT

Sample output

**Brand:** Forma Studio (minimal architecture firm) **Core suffix** (append to every prompt): `--style raw --ar 16:9 --v 6.1 :: warm ivory and slate tones, natural light, architectural precision, editorial calm` **Approved styles:** 1. Exterior — `concrete and glass building, golden hour, aerial perspective, Zaha Hadid influence` 2. Interior — `open plan living space, floor-to-ceiling windows, linen furniture, minimal decor` ... **Always avoid:** busy patterns, bright saturated colors, people in foreground, text overlays, fantasy elements **Color palette to include:** ivory (#F7F4EE), slate (#4A5568), warm sand (#C9B99A)
