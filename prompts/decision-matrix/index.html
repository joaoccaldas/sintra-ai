The prompt

Help me make a difficult decision using a structured decision matrix. Decision to make: [describe the choice you're facing clearly] Options available: [list all options you're considering — at least 2, ideally 3–5] What matters most to me: [list 4–8 criteria that this decision should be evaluated against] Constraints: [anything that disqualifies an option — budget, time, dependencies] Who else is affected: [list any stakeholders whose interests matter] Deadline to decide: [when must this be decided?] Gut leaning (if any): [your current instinct, or 'none'] Deliver: 1. A weighted decision matrix — assign weights to each criterion (totaling 100%) based on what you told me, then score each option 1–5 per criterion 2. Final weighted scores and a ranking of options 3. Sensitivity analysis — which criterion, if weighted differently, would change the top choice? 4. The strongest argument FOR each option (steelman) 5. The biggest hidden risk in the top-ranked option 6. A recommended decision with a 3-sentence rationale 7. A reversibility assessment — how hard is it to undo each option? (1=easy to reverse, 5=permanent) 8. The one question you should answer before committing, if any uncertainty remains
