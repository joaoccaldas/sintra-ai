Financial Modeling with AI

A complete 3-statement model spec: structure, drivers, sensitivities, dashboard, audit checks, and documentation.

Recommended modelGPT-5.5 Instant

Excellent tabular logic and financial model structuring

What you need to fill in

[purpose: M&A/new product/investment][blank template / partial model / needs review][Excel/Google Sheets]

Tools used

ClaudeExcelGoogle Sheets

The prompt

"I'm building a financial model for [purpose: M&A/new product/investment]. Requirements: - 3-statement model (P&L, BS, CF) - 5-year projection - Sensitivity analysis - Key assumptions clearly documented Current stage: [blank template / partial model / needs review] Please: 1. Structure the model layout 2. Build core formulas and drivers 3. Create assumption tables 4. Add sensitivity tables (±10%, ±20%) 5. Build summary dashboard 6. Add error checks and validations 7. Write documentation Format: [Excel/Google Sheets] Include: - Color coding scheme - Formula explanations - Circularity handling - Audit trail"
Open in ChatGPT

Sample output

**Tabs** 1. `Cover` — purpose, version, owner 2. `Drivers` — colour-coded inputs (blue) 3. `P&L`, `BS`, `CF` — links only (black), no hardcodes 4. `Sensitivity` — ±10/20% on revenue growth × GM 5. `Checks` — 6 integrity rules (BS balances, CF reconciles) **Driver math (revenue)** ``` Rev_t = Cust_t × ARPU_t Cust_t = Cust_{t-1} × (1 − Churn_t) + NewLogo_t ``` **Checks:** all return PASS or the cell turns red. Circularity resolved with iterative calc, max 100, threshold 0.001.
