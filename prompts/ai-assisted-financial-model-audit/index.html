AI-Assisted Financial Model Audit

An audit report with severity-tagged findings: assumption consistency, formula integrity, broken links, anomalies, and improvements.

⬡
Recommended modelClaude Sonnet 4.6

Handles complex queries across large datasets

What you need to fill in

[DCF/3-statement/budget][years][list][locations]

Tools used

ClaudeExcel

The prompt

"Audit my financial model for consistency and errors. Model details: - Purpose: [DCF/3-statement/budget] - Time horizon: [years] - Key assumptions: [list] - Check cells: [locations] Please: 1. Verify assumption consistency and sign 2. Check formula links and circularities 3. Validate check cells = 0 or PASS 4. Identify broken links or #REF errors 5. Flag unusual patterns (e.g., margins jumping 50%) 6. Suggest improvements Output: Audit report with severity ratings (Critical/Warning/Info)"
Open in Claude

Sample output

**Critical (3)** - `WACC!C18` references a deleted range — #REF! - BS doesn't balance in years 4–5: $1.2M, $0.9M off - Tax rate hard-coded inside a formula in CF tab **Warnings (7)** - Revenue growth jumps 14% → 41% Y3 with no driver explanation - Two circularities resolved silently — flag for review - Three named ranges declared but unused **Info (12)** — naming, colour coding, font drift. **Recommendation:** fix criticals before sending; warnings before signing.
