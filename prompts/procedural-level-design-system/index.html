**System: Dungeon Crawler Level Generator** **Grammar rules:** Room types: [ENTRANCE, COMBAT, TREASURE, PUZZLE, BOSS] Sequence: ENTRANCE → (COMBAT|PUZZLE)+ → TREASURE? → BOSS Constraint: No two COMBAT rooms adjacent; TREASURE only after COMBAT **Difficulty curve:** Enemy HP = base × (1 + 0.15 × roomIndex) Room count = 8 + floor(level/3) **Pseudocode:** ``` function generateLevel(seed, difficulty): rng = SeededRandom(seed) rooms = [ENTRANCE] while not valid(rooms): rooms = buildChain(rng, difficulty) return connectRooms(rooms, rng) function validate(rooms): return isReachable(rooms) and hasBoss(rooms) and difficultyInBounds(rooms) ``` **Sample output (seed 4821, difficulty 2):** ENTRANCE → COMBAT(3 goblins) → PUZZLE(pressure plates) → COMBAT(5 goblins + archer) → TREASURE(rare sword) → BOSS(Lich, 240HP)
