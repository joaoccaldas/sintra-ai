Sample output

**Templates** 1. **Acknowledgement** — "Got it, [name]. Looking into it now — back to you within 2 hours." 2. **Asking for more info** — "Quick thing to unblock me: could you send [X, Y]?" 3. **Known issue** — "Yep, known one — fix is going out [date]. Workaround: [...]" 4. **Escalation** — "Pulling in [team]. Owner is [name]. Updates in this thread." 5. **Resolution** — "[Fix] is live. Anything else from this ticket?" **Flowchart** (login issue): Reset password? → Y: send link / N: 2FA enrolled? → Y: code working? → escalate.
