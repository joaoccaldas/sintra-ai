Ask Your Power BI Dashboard Questions in Plain English

An exact natural-language query for each of your business questions, the expected visual type, a drill-down follow-up, and a refinement tip per question.

The prompt

"I have a Power BI dashboard and I want to use Copilot's Q&A feature to explore data with natural language instead of building visuals manually. My dashboard covers: [topic, e.g. sales performance / HR headcount / inventory levels] Key measures available: [list 3–5, e.g. Revenue, Headcount, Stock Levels] I typically need to answer these business questions: [list 3–5 real questions your team asks] For each of my questions, write: 1. The exact natural-language query to type into Power BI Copilot / Q&A 2. What type of visual it will likely return 3. A follow-up question to drill deeper 4. One tip to refine the answer if the first result is off"
Open in ChatGPT

Sample output

**Q1: 'Are we hitting our sales target?'** - Query to type: *"Show revenue vs target by region this quarter as a bar chart"* - Expected visual: Clustered bar — Actual vs Target side by side per region - Follow-up: *"Which sales rep in the bottom region has the largest gap?"* - Refinement tip: If Copilot uses the wrong date range, add *"for Q3 FY2026"* explicitly **Q2: 'Which product is selling the most?'** - Query to type: *"Top 10 products by units sold this year"* - Expected visual: Ranked bar chart, descending - Follow-up: *"Show the same top 10 by revenue margin instead of units"* - Refinement tip: Say *"by product name not SKU"* if it returns codes instead of readable names **Q3: 'How is this month trending vs last month?'** - Query to type: *"Daily revenue this month vs last month as a line chart"* - Expected visual: Dual-line chart with month comparison - Follow-up: *"Highlight the days where this month fell below last month"* - Refinement tip: If it shows weekly not daily, add *"daily granularity"* to your prompt
