May 2026Major

Anthropic Releases Claude Opus 4.8 — 69.2% SWE-Bench Pro, 2.5× Faster

Anthropic released Claude Opus 4.8 on May 28, available on Claude API, Amazon Bedrock, Google Cloud Vertex AI, and Microsoft Foundry. Agentic coding score improved from 64.3% to 69.2% on SWE-Bench Pro, knowledge-work score jumped from 1,753 to 1,890, and the new fast mode is 2.5× faster at 3× lower cost. New capabilities include Dynamic Workflows (parallel subagent orchestration) and an effort-control toggle. Pricing remains $5/$25 per million tokens.

May 2026Major

Microsoft Build 2026: Proprietary Coding Model, MCP Standard, Agentic AI Focus

Microsoft announced at Build 2026 (May 28) a new suite of specialised AI models for transcription, reasoning, voice, and image processing, plus a proprietary coding model targeting Cursor and Claude Code at lower enterprise cost. The conference centred on agentic AI, with MCP named as the preferred inter-agent standard for Azure AI, AutoGen v2, and Visual Studio Code Copilot updates. Microsoft also announced multi-agent orchestration natively in Microsoft 365 Copilot for enterprise workflows.

May 2026Major

Microsoft Copilot Studio: Computer-Using Agents Now Generally Available

Microsoft's May 2026 Copilot Studio update made computer-using agents generally available in all commercial regions, enabling agents to interact with desktop and web applications exactly as a human would — clicking, typing, and navigating. The update also introduced redesigned Workflows, Work IQ extensibility, and real-time voice. Agent 365, a dedicated IT console for deploying and governing AI agents, reached general availability at $15/user/month.

May 2026Notable

GitHub Copilot Switches to Usage-Based AI Credits on June 1 — Agentic Sessions Made Flat Fees Unsustainable

GitHub announced all Copilot plans will transition from flat-fee, request-based billing to usage-based 'AI Credits' ($0.01 per credit) on June 1, 2026, as agentic multi-step coding sessions made the previous unlimited model economically unsustainable. Copilot Pro+ ($39/month) includes $39 in monthly credits; Business ($19/user/month) includes $19; enterprise tiers scale proportionally. Annual subscribers remain on request-based billing until plan expiry, and credit overage is billed at the same $0.01 rate.

May 2026Major

SpaceX IPO Filing Reveals Anthropic Paying $1.25B/Month for xAI's Colossus GPU Cluster — $45B Total

SpaceX's IPO prospectus disclosed that Anthropic is paying xAI $1.25 billion per month through May 2029 for exclusive access to the Colossus 1 data center in Memphis, Tennessee, covering over 300 megawatts and more than 220,000 Nvidia GPUs — totalling approximately $45 billion over the contract term. xAI retains the Colossus 2 facility for its own Grok models. Both parties hold a 90-day termination option, and capacity ramped up in May–June 2026 at an initial discounted rate.

May 2026Notable

OpenAI Codex Can Now Control a Locked Mac via Apple Authorization Plugin

OpenAI added 'Locked Computer Use' to Codex — an Apple authorization plugin that temporarily unlocks a Mac with strict temporal and behavioral safeguards, enabling developers to trigger and monitor long-running agent tasks remotely from their phone even when the machine's screen is locked. The feature, rolling out to eligible Mac users, is intended for overnight coding runs and CI-style autonomous loops without requiring the developer to be physically present at the machine. Windows support has not been announced.
