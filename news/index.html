Anthropic

Claude Fable 5 — Anthropic's Most Capable Model Yet, 95% on SWE-bench Verified

Anthropic released Claude Fable 5 on June 9, 2026, alongside the restricted Claude Mythos 5, describing it as its most capable generally-available model to date. Fable 5 scores roughly 95% on SWE-bench Verified and 92.6% on GPQA Diamond, ships with a 1M-token context window and up to 128K output tokens, and runs with always-on adaptive thinking — the legacy 'disable thinking' option is no longer supported. API pricing is $10 per million input tokens and $50 per million output tokens, with new safety classifiers that can return a 'refusal' stop reason for dual-use cyber, bio, and chem topics and automatically fall back to Claude Opus 4.8.

Why it mattersA 95% SWE-bench Verified score with a 1M-token context window and built-in self-verification pushes the ceiling for what a single model can autonomously accomplish on multi-day software engineering and research tasks.

Try itRun Claude Fable 5 on your hardest unresolved coding ticket via Claude Code with the memory tool enabled, and compare completion quality and autonomy against Opus 4.8 on the same task.
